What I've been doing and some short term plans

2025/06/29

What I've been doing, reading and listening to recently

For the last month I've started to work on a new project called gbc. I'm trying to make my own implementation of gradient boosting in C. I'm currently reading the book Classification and Regression Trees published in 1984 by Breiman et al. It explains the methods used to create trees efficiently in computer programs. I hope to finish this program in the next week or two so that I can move on to my next project. This next project I'm eager to work on is a chess engine. I hope to learn how neural networks work as it feels like a natural progression from gradient boosting. I also really like chess, I'm not a very good player though. Another thing I plan to do, not necessarily after these projects, is to film some videos explaining how ss-py works. I also plan on making a quick tour of the website where all of the data from the models is currently hosted. In the longer term I hope to create a website to host the data from the baseball models I create like lichess, it being free and open source with no ads and no subscriptions. I also need to update cpi-csi and its website.

Asides from this, I've been re-reading for the third time Anna Karenina, it is my favourite book. I think it is the kind of book where every time you read it, you'll pick up different things, and your opinions on events and characters can change from read to read.

I've been working out as always. I'm now doing exercises like hand stands and pike push ups to be able to do hand stand push ups. I hope these will help me grow my shoulders. I also think that in the near future I will be buying weights so that I can do weighted dips and pull ups.
